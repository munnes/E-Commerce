E-Commerce

Project Definition:

E-Commerce project is combination of web and mobile applications that serve:

  • Registered venders
  • Venders’ customers
  • Delivery company

How it works, in few words?

  • Venders register in the application, add their items and mange customers’ orders.
  • Delivery Company manage venders , drivers and venders’ customers orders
  • Venders’ Customers can purchase their items throw web application and mobile application
  • Drivers receive orders throw mobile application and update package status

Technologies used:

  • Build web application front end using React js
  • Build customer and driver mobile applications front end using React Native
  • Use Node js and express server as backend
  • Use MySql as database
  • Deal with third parties such as Google Maps APIs
